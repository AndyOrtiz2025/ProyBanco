Este proyecto es una simulación de un sistema de atención en un banco que utiliza una cola (FIFO - First In, First Out) para gestionar el orden de llegada de los clientes.
La aplicación permite agregar clientes, atenderlos en orden y visualizar la cantidad de clientes en espera en todo momento. Además, incluye una funcionalidad opcional para manejar clientes prioritarios (como personas mayores o con discapacidad).

COMO USARLO:
Ingresa la opción que deseas y sigue las instrucciones:

Opción 1: Ingresa el nombre del cliente y si es prioritario (si para sí, cualquier otra tecla para no).
Opción 2: Atiende al primer cliente en la cola y muestra cuántos quedan en espera.
Opción 3: Muestra la lista de clientes en espera.
Opción 4: Salir del sistema.
